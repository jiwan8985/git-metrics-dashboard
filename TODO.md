- **심각도:** 🔴 CRITICAL (사용자 저장소 보안 위협)

  ```typescript

    ```
  - [ ] **방안 2:** 배열 기반 명령어 실행 (Node.js spawn)
    ```typescript
    import { spawn } from 'child_process';
    const git = spawn('git', ['log', '--since=' + sinceStr, ...]);
    ```
  - [ ] sinceStr 형식 엄격한 정규식 검증
  - [ ] Output sanitization (XSS 방지)
- **예상 시간:** 4-6시간
